## 🎯 Présentation du Projet

### Contexte
Application backend complète développée avec **Spring Boot** pour la gestion centralisée d'une compagnie d'assurance. Elle permet de gérer l'ensemble du cycle de vie des clients, contrats, paiements et sinistres.

### Objectifs
- ✅ **Centraliser** la gestion des assurés et leurs contrats
- ✅ **Automatiser** le suivi des paiements et échéances
- ✅ **Optimiser** le traitement des sinistres
- ✅ **Fournir** des statistiques en temps réel
- ✅ **Garantir** la traçabilité des opérations

### Public Cible
- Compagnies d'assurance
- Courtiers en assurance
- Agents généraux
- Gestionnaires de sinistres

## ✨ Fonctionnalités

### 👥 Gestion des Clients
| Fonctionnalité | Description |
|----------------|-------------|
| CRUD Complet | Création, lecture, mise à jour, suppression |
| Recherche avancée | Par nom, email, téléphone |
| Pagination | Navigation optimisée dans les listes |
| Statistiques | Nombre de contrats, total cotisations, taux fidélité |
| Historique | Traçabilité des opérations |

### 📄 Gestion des Contrats
#### Types de Contrats Supportés

| Type | Caractéristiques |
|------|------------------|
| 🚗 **Automobile** | Immatriculation, marque, modèle, année, puissance fiscale, usage |
| 🏠 **Habitation** | Type logement, superficie, pièces, alarme, type chauffage |
| 🏥 **Santé** | Niveau couverture, nb personnes, options optique/dentaire |
